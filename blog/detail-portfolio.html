Membuat Website Portfolio

Dipublikasikan: Juli 2025

Dalam blog ini, saya membagikan proses saya membangun website portfolio pribadi menggunakan HTML, CSS, dan Bootstrap 5. Mulai dari merancang layout, memilih warna, membuat responsive, hingga deploy ke hosting seperti Railway atau GitHub Pages.

Hal terpenting adalah konsistensi desain, pemilihan font yang bersih, dan kejelasan informasi tentang diri saya serta karya-karya saya.

Tools yang saya gunakan:

  • VS Code
  • Bootstrap 5
  • Icons dari Bootstrap & FontAwesome
