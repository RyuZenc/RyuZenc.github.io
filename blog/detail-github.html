Tips Git & Github

Dipublikasikan: Mei 2025

Menggunakan Git & GitHub sangat penting untuk semua developer. Di awal, saya belajar cara inisialisasi repository, commit perubahan, dan membuat branch untuk fitur baru.

Tips favorit saya:

  • Gunakan `.gitignore` untuk file yang tidak perlu
  • Selalu beri nama branch yang jelas (contoh: `feature/blog-page`)
  • Pakai Pull Request untuk review sebelum merge

Saya juga belajar tentang GitHub Pages untuk hosting portfolio statis secara gratis.
